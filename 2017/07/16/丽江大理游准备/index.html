丽江，大理城市交通

  1. 19号 14.20~19.15,飞机到达丽江，公交2.5小时，打的1小时
  2. 乘坐公交27路，约2小时到达古城客栈，距离29公里，可叫专车或者的士
  3. 21号9.30丽江客运站乘大巴至大理，车程3.5个小时(古城至客运站步行1.5公里，约20分钟)
  4. 大理下关北客运站，距酒店1.5公里，步行约20分钟，根据天气可考虑打的(大约下午1点左右到)
  5. 24号，打车至机场，约18公里，车程50分钟，11点飞机(8.30出发)

丽江景点游玩

玉龙雪山

费用门票

景区门票(进山票￥130)+环保车票（￥20）+大索道票（￥180）

羽绒服（￥100）+氧气大瓶（￥100）

门票紧张需提前预定，根据天气头一天晚上则需在网上预定

交通

  1. 每天有1~2班的旅游巴士从古城北门口出发，单程车票35，来回车票60，车程约1小时（具体时间咨询一下当地人，客栈掌柜）
  2. 打车约19公里，约40分钟

描述

  • 进入景区内后，统一到甘海子然后乘坐大巴车至蓝月谷或者缆车站，可选择先去蓝月谷浏览，亦可在下山后浏览，蓝月谷有云南小九寨之称
  • 缆车乘车点海拔3369，总上升海拔1千米以上，到达4506海拔处观景台，可步行至山顶，下缆车时注意高反，氧气瓶还是备一个为好

重点

  • 携带防晒霜
  • 山上气温会较冷
  • 天气多变，携带雨伞，遮阳伞
  • 天气变化体感温度变化，携带外套
  • 防高反的药
  • 饮用水

一日游散团

  • （携程）官方直营套票（￥660），包含丽江至雪山旅游巴士往返，进山票，环保车票，索道票，雪厨48元餐一张，羽绒服，氧气，矿泉水（游玩前1天22:00前预定，早8点出发）
  • （去哪儿）豆豆假期散团（￥498），同上全包（游玩前1天23:00预定，早8点出发）
  • （蚂蜂窝）一日游（￥509），同上全包（游玩前1天19:00前购买，早7:30出发）

拉市海

  • 湿地公园
  • 茶马古道骑马
  • 划船

交通

  • 古城步行至古城区政府站，公交1、11路至玉龙县医院站下车（5站），换乘公交31、32路至道班房站下车（8站），下车后回行望海酒楼路口往大丽高速方向步行约400米，总行程约1.2小时
  • 如果天气理想，可租摩托车电瓶车前往
  • 打车，全程约12公里，走大丽高速，总行程约25分钟

重点

  • 防晒
  • 询价游玩
  • 饮用水，食物
  • 如需骑马，着装注意

一日游散团

  • （蚂蜂窝）一日游（￥168）包含来回交通、午餐、茶马古道线路、皮划艇不限时，可选100元的沙滩摩托1小时

大理景点游玩

洱海

  • 喜洲古镇（西线）
  • 海舌公园(开放时间：6:30——18:30)
  • 双廊镇（双廊派出所日落观看点）、南诏风情岛（东线）(整改)
  • 挖色镇（东线）

交通

  • 电瓶车，洱海东西环线130公里，需半途充电，充一次电10元，且车速较慢，游玩一圈可能需要一整天有余的时间
  • 摩托车，如遇天气不佳，下雨什么的，不适用，但体验最好，随停随走
  • Smart汽车，交通复杂地方，有错车问题，环海线上可能也不是所有的地方都能方便随时停车
  • 环海西路+环海东路(西线整改)

描述

  • 洱海环线130公里，建议顺时针游玩（西线至东线）
  • 双廊17年9月前都在环保整改，大批店面关张，道路也在维修，路况较差，可能会尘土飞扬
  • 根据具体的游玩节奏，可选2天游玩，一天西线，一天东线，以双廊为终点返程
  • 西线的日出，海东路的晚霞，时间允许的前提下可以一看（晚霞一般在19~20点）

重点

  • 防晒
  • 拍照
  • 带饮用水，零食
  • 环海地图+GPS导航
